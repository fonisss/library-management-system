# 简单图书馆管理系统

## 项目简介

这是一个使用C++编写的简单图书馆管理系统。它旨在提供一个基础的图书管理功能，包括添加图书、删除图书、查找图书和列出所有图书。

## 开发环境

- **编程语言**：C++
- **编译器**：推荐使用 GCC 或 Clang
- **开发平台**：项目可在 Windows、Linux 和 macOS 上编译和运行。

## 构建与运行

### Windows

1. 确保安装了 Visual Studio 或 MinGW。
2. 使用以下命令编译代码：

   ```bash
   g++ -o LibrarySystem main.cpp
